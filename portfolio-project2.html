Abstract

  • Data: 2018
  • Tempo di lettura: 3 min.

Our project is called SCARA Robot. It is a robot composed of four axis. In a horizontal plane two articulated arms move, hinged at one end while on the other free end, there is a Z axis, which can move vertically. It is a robot used for a movement called Pick & Place because it can operate at significantly higher speeds and it can work in operations where precision and accuracy are essential. Mathematical laws called inverse kinematics regulate its movement. Starting from the point that you want the robot to go, you can calculate, thanks to these formulas, the angles of the two motors. When the robot arrives in the final position the Z-axis goes down to hold things with an industrial sucker. We used a complex control system, made of a Texas Instruments board communicating with a C++ application running on a computer, which implemented the autonomous mode, an Android application that implemented manual mode and a C++ application running on another computer, which implemented the vision system.
